# Trigonometry

## Basic Definitions

### Trigonometric Ratios
In a right triangle with angle θ:
- sin θ = opposite/hypotenuse
- cos θ = adjacent/hypotenuse
- tan θ = opposite/adjacent
- cot θ = adjacent/opposite = 1/tan θ
- sec θ = hypotenuse/adjacent = 1/cos θ
- csc θ = hypotenuse/opposite = 1/sin θ

### Fundamental Relations
1. sin²θ + cos²θ = 1
2. 1 + tan²θ = sec²θ
3. 1 + cot²θ = csc²θ
4. tan θ = sin θ/cos θ
5. cot θ = cos θ/sin θ

## Standard Angles

| Angle | 0° | 30° | 45° | 60° | 90° |
|-------|-----|-----|-----|-----|-----|
| sin   | 0   | 1/2 | 1/√2 | √3/2 | 1   |
| cos   | 1   | √3/2 | 1/√2 | 1/2 | 0   |
| tan   | 0   | 1/√3 | 1   | √3  | ∞   |

## Trigonometric Identities

### Pythagorean Identities
1. sin²A + cos²A = 1
2. 1 + tan²A = sec²A
3. 1 + cot²A = csc²A

### Angle Sum and Difference
1. sin(A ± B) = sin A cos B ± cos A sin B
2. cos(A ± B) = cos A cos B ∓ sin A sin B
3. tan(A ± B) = (tan A ± tan B)/(1 ∓ tan A tan B)

### Double Angle Formulas
1. sin 2A = 2 sin A cos A
2. cos 2A = cos²A - sin²A = 2cos²A - 1 = 1 - 2sin²A
3. tan 2A = 2 tan A/(1 - tan²A)

### Triple Angle Formulas
1. sin 3A = 3 sin A - 4 sin³A
2. cos 3A = 4 cos³A - 3 cos A
3. tan 3A = (3 tan A - tan³A)/(1 - 3 tan²A)

### Half Angle Formulas
1. sin(A/2) = ±√[(1 - cos A)/2]
2. cos(A/2) = ±√[(1 + cos A)/2]
3. tan(A/2) = sin A/(1 + cos A) = (1 - cos A)/sin A

### Product to Sum Formulas
1. 2 sin A cos B = sin(A+B) + sin(A-B)
2. 2 cos A sin B = sin(A+B) - sin(A-B)
3. 2 cos A cos B = cos(A+B) + cos(A-B)
4. 2 sin A sin B = cos(A-B) - cos(A+B)

### Sum to Product Formulas
1. sin A + sin B = 2 sin[(A+B)/2] cos[(A-B)/2]
2. sin A - sin B = 2 cos[(A+B)/2] sin[(A-B)/2]
3. cos A + cos B = 2 cos[(A+B)/2] cos[(A-B)/2]
4. cos A - cos B = -2 sin[(A+B)/2] sin[(A-B)/2]

## Trigonometric Equations

### General Solutions
1. sin θ = sin α ⟹ θ = nπ + (-1)ⁿα, n ∈ Z
2. cos θ = cos α ⟹ θ = 2nπ ± α, n ∈ Z
3. tan θ = tan α ⟹ θ = nπ + α, n ∈ Z

### Principal Solutions
Solutions in the range [0, 2π] or [0°, 360°]

## Inverse Trigonometric Functions

### Domains and Ranges
1. sin⁻¹x: Domain [-1,1], Range [-π/2, π/2]
2. cos⁻¹x: Domain [-1,1], Range [0, π]
3. tan⁻¹x: Domain ℝ, Range (-π/2, π/2)
4. cot⁻¹x: Domain ℝ, Range (0, π)
5. sec⁻¹x: Domain (-∞,-1]∪[1,∞), Range [0,π]-{π/2}
6. csc⁻¹x: Domain (-∞,-1]∪[1,∞), Range [-π/2,π/2]-{0}

### Properties
1. sin⁻¹(-x) = -sin⁻¹x
2. cos⁻¹(-x) = π - cos⁻¹x
3. tan⁻¹(-x) = -tan⁻¹x
4. sin⁻¹x + cos⁻¹x = π/2
5. tan⁻¹x + cot⁻¹x = π/2
6. sec⁻¹x + csc⁻¹x = π/2

### Addition Formulas
1. sin⁻¹x + sin⁻¹y = sin⁻¹[x√(1-y²) + y√(1-x²)]
2. tan⁻¹x + tan⁻¹y = tan⁻¹[(x+y)/(1-xy)], if xy < 1
3. tan⁻¹x - tan⁻¹y = tan⁻¹[(x-y)/(1+xy)]

## Properties of Triangles

### Sine Rule
a/sin A = b/sin B = c/sin C = 2R

Where R is the circumradius

### Cosine Rule
1. a² = b² + c² - 2bc cos A
2. b² = a² + c² - 2ac cos B
3. c² = a² + b² - 2ab cos C

Also: cos A = (b² + c² - a²)/(2bc)

### Projection Formula
1. a = b cos C + c cos B
2. b = c cos A + a cos C
3. c = a cos B + b cos A

### Half-Angle Formulas
1. sin(A/2) = √[s(s-a)/(bc)]
2. cos(A/2) = √[s(s-a)/(bc)]
3. tan(A/2) = √[(s-b)(s-c)/(s(s-a))]

Where s = (a+b+c)/2

### m-n Theorem
(m + n)cot θ = m cot A - n cot B

Where θ is angle between median and side

## Heights and Distances

### Angle of Elevation
Angle made with horizontal when looking up

### Angle of Depression
Angle made with horizontal when looking down

### Important Relations
In problems involving heights:
- tan θ = height/distance
- Use right triangle properties
- Consider multiple triangles for complex problems

## Graphs of Trigonometric Functions

### Sine Function
- Period: 2π
- Amplitude: 1
- Range: [-1, 1]
- Symmetric about origin (odd function)

### Cosine Function
- Period: 2π
- Amplitude: 1
- Range: [-1, 1]
- Symmetric about y-axis (even function)

### Tangent Function
- Period: π
- Range: (-∞, ∞)
- Asymptotes at x = ±π/2, ±3π/2, ...
- Odd function

## Radians and Degrees

### Conversion
- π radians = 180°
- 1 radian = 180°/π ≈ 57.3°
- 1° = π/180 radians ≈ 0.01745 radians

### Arc Length
s = rθ (θ in radians)

### Sector Area
A = (1/2)r²θ (θ in radians)
