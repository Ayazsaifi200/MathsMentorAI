# Algebra Fundamentals

## Linear Equations

### Standard Form
A linear equation in one variable: ax + b = 0, where a ≠ 0

Solution: x = -b/a

### Linear Equations in Two Variables
Standard form: ax + by + c = 0

Slope-intercept form: y = mx + c
- m is the slope
- c is the y-intercept

Point-slope form: y - y₁ = m(x - x₁)

## Quadratic Equations

### Standard Form
ax² + bx + c = 0, where a ≠ 0

### Quadratic Formula
x = (-b ± √(b² - 4ac)) / (2a)

### Discriminant
D = b² - 4ac
- D > 0: Two distinct real roots
- D = 0: One repeated real root
- D < 0: No real roots (complex roots)

### Sum and Product of Roots
If α and β are roots:
- Sum: α + β = -b/a
- Product: αβ = c/a

### Factorization Methods
1. Common factor
2. Grouping
3. Splitting the middle term
4. Difference of squares: a² - b² = (a+b)(a-b)
5. Perfect square: a² + 2ab + b² = (a+b)²

## Polynomials

### General Polynomial
P(x) = aₙxⁿ + aₙ₋₁xⁿ⁻¹ + ... + a₁x + a₀

### Remainder Theorem
When P(x) is divided by (x - a), remainder = P(a)

### Factor Theorem
(x - a) is a factor of P(x) if and only if P(a) = 0

### Vieta's Formulas
For polynomial xⁿ + a₁xⁿ⁻¹ + ... + aₙ = 0 with roots r₁, r₂, ..., rⁿ:
- Sum of roots: r₁ + r₂ + ... + rⁿ = -a₁
- Product of roots: r₁ × r₂ × ... × rⁿ = (-1)ⁿaⁿ

## Inequalities

### Properties
1. If a < b and c > 0, then ac < bc
2. If a < b and c < 0, then ac > bc
3. If a < b, then a + c < b + c

### AM-GM Inequality
For positive numbers a and b:
(a + b)/2 ≥ √(ab)

Equality holds when a = b

### Triangle Inequality
|a + b| ≤ |a| + |b|

## Sequences and Series

### Arithmetic Progression (AP)
General term: aₙ = a + (n-1)d
- a: first term
- d: common difference

Sum of n terms: Sₙ = n/2 [2a + (n-1)d] = n/2 [a + l]
- l: last term

### Geometric Progression (GP)
General term: aₙ = arⁿ⁻¹
- r: common ratio

Sum of n terms: Sₙ = a(rⁿ - 1)/(r - 1), when r ≠ 1

Infinite GP sum: S∞ = a/(1 - r), when |r| < 1

## Binomial Theorem

### Binomial Expansion
(a + b)ⁿ = Σ(k=0 to n) C(n,k) aⁿ⁻ᵏ bᵏ

Where C(n,k) = n!/(k!(n-k)!)

### Important Properties
1. Total terms = n + 1
2. Sum of binomial coefficients = 2ⁿ
3. Middle term: (n/2 + 1)th term when n is even

## Logarithms

### Fundamental Properties
1. log_a(mn) = log_a(m) + log_a(n)
2. log_a(m/n) = log_a(m) - log_a(n)
3. log_a(mⁿ) = n log_a(m)
4. log_a(a) = 1
5. log_a(1) = 0

### Change of Base
log_b(x) = log_a(x) / log_a(b)

### Natural Logarithm
ln(x) = log_e(x), where e ≈ 2.71828
