# Step-by-Step: Get Groq API Key

## 1. Open Groq Console
Go to: https://console.groq.com

## 2. Sign Up (FREE)
Click "Sign in" button (top right)
Options:
- Sign in with Google (recommended - fastest)
- Sign in with GitHub
- Or create account with email

## 3. Create API Key
After login:
1. You'll see the Groq console dashboard
2. Look for "API Keys" in the left sidebar
3. Click "Create API Key"
4. Give it a name: "Math Mentor AI"
5. Click "Submit"

## 4. Copy Your Key
You'll see a key starting with: gsk_...
Example: gsk_abc123xyz...

⚠️ IMPORTANT: Copy it NOW - you can only see it once!

## 5. Save to .env file
Open: d:\MathMentorAIprojectAssesment\.env
Add this line:

GROQ_API_KEY=gsk_your_actual_key_here

Save the file!

## DONE! 🎉
Your Math Mentor is now ready to use Groq's powerful Llama 3.3 70B model!
